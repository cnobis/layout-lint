Demo catalog

Layout-lint is a browser-first DSL for validating layout relationships against real DOM geometry.

This page is the staged demo track used to develop and verify language features.

Stage 1 — Gallery
Baseline constraints demo with live overlays; tests vertical flow, overlap, and two-column relations.
Stage 2 — Control Room
Interactive command-deck demo; tests `centered_x` and equal-gap constraints with live spacing controls.
Stage 3 — Dashboard
Panel and card composition stage; tests grid rhythm, card adjacency, and cross-section spacing.
Stage 4 — Forms
Form layout stage; tests label-input alignment, section grouping, and action-row consistency.
